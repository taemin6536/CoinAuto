# 트레이딩 전략 가이드

이 문서는 업비트 자동매매 봇에서 사용할 수 있는 트레이딩 전략들에 대한 설명과 사용법을 제공합니다.

## 지원되는 전략

### 1. 단순 이동평균 교차 전략 (SMA Crossover)
### 2. RSI 모멘텀 전략 (RSI Momentum)

## 전략 설정 방법

1. `config/strategies/` 디렉토리에서 원하는 전략의 YAML 파일을 수정
2. 봇을 재시작하거나 핫 리로드 기능 사용
3. 로그를 통해 전략 동작 확인

## 전략 개발 가이드

새로운 전략을 개발하려면:

1. `upbit_trading_bot/strategy/base.py`의 `TradingStrategy` 클래스를 상속
2. `evaluate()` 메서드 구현
3. 설정 파일 생성
4. 테스트 작성

자세한 내용은 각 전략별 문서를 참조하세요.
